WURFL Device Capabilities

Useful information developers can detect from a device

Capability Explanation Why It Helps Developers
brand_name This capability identifies the brand of the device, such as Apple, Samsung, or Google. It helps developers understand what kind of device is requesting the application and can support device-specific testing and optimization.
model_name This capability identifies the exact model name of the device. It allows developers to distinguish between devices from the same brand. This allows for better support, optimization, and debugging.
resolution_width This capability gives the width of the device screen in pixels. It helps developers decide how content should be displayed so the interface fits properly on different screen sizes.
