Template Selection, Local Storage, and Basic PWA Features

Progressive Web Application

A Progressive Web Application, or PWA, is a web application that behaves more like an installed application. A basic PWA usually includes a web app manifest, an icon, HTTPS hosting, and a service worker. These features allow the browser to recognize the site as installable and make it possible for users to launch it from their desktop or home screen.

One of the most important characteristics of a PWA is reliability. A PWA can continue working even when the user is offline because a service worker can cache important files, such as the HTML page, CSS and JavaScript files, and images. This makes the application faster on later visits and allows the user to access the application offline instead of seeing a blank page.
